# Бот "Что едим сегодня?"

Телеграм-бот для учёта мясных заготовок и ежедневного выбора блюда мужем.

## 1. Получить токен бота

1. Напиши в Telegram боту [@BotFather](https://t.me/BotFather)
2. Команда `/newbot`, следуй инструкциям — получишь токен вида `123456:AA...`

## 2. Узнать свой chat_id и chat_id мужа

## 4. Команды бота

**Для жены (админ):**
- `/add` — добавить заготовку мяса, гарнир или салат
- `/list` — посмотреть текущие остатки
- `/edit` — изменить количество и/или название заготовки
- `/delete` — удалить позицию
- `/settime ЧЧ:ММ` — во сколько присылать мужу вопрос каждый день
- `/setgarnish` — если для заготовки подходят только определенные гарниры, можно их изменить тут
- `/setcancel` <часы> <минуты>	Установить время, в течение которого муж может изменить выбор
- `/showcancel` Показать текущее время на изменение выбора
- `/ask` — отправить вопрос мужу прямо сейчас, не дожидаясь расписания
- `/help` — все команды

**Для мужа:**
- Отвечает на кнопки, когда бот сам присылает вопрос
- `/food` — можно запросить вопрос вручную, если бот почему-то не написал сам
- Кнопка "🚚 Доставка" — если не хочется готовки, можно попросить заказать еду;
  бот спросит, что именно, и перешлёт это жене (остатки заготовок при этом не трогаются)

## Особенности логики

- Мясные заготовки — хранятся с количеством порций. При выборе мужем списывается 1 порция.
- Гарниры и салаты не имеют учёта количества — считаются всегда доступными
- У мясной заготовки можно один раз при добавлении зафиксировать гарнир
  по умолчанию (например, у плова — рис). Тогда при выборе этого блюда шаг
  выбора гарнира просто пропускается
- При подтверждении выбора количество мяса уменьшается на 1, а жене
  приходит итоговое сообщение. Если заготовка закончилась — жена получит
  дополнительное предупреждение
- Муж может изменить свой выбор в течение 4 часов по дефолту, или в течение заданного женой времени, в таком случае мясная заготовка, выбранная изначально, будет возвращена в базу, и списана вновь выбранная
